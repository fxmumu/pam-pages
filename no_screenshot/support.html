2. Features Overview

OCR (Text Recognition): Extract text from screenshots or photos. Processed 100% locally using Apple Vision Framework. No internet required.

LLM Text Enhancement: Use credits to format, summarize, or organize text. Requires internet connection. Only text is sent to cloud, never images.

Semantic Search: Find related notes by meaning. Processed locally using CoreML.

3. Common Questions

Q: Is my data safe?

A: Your screenshots, OCR text, and notes are stored locally on your device. We cannot access them. Only when you use LLM features, the relevant text is sent to cloud for processing.

Q: Do I need internet to use the app?

A: Basic features (OCR, note storage, search) work offline. Only LLM text enhancement requires internet.

Q: How do credits work?

A: Credits are used for LLM features. You get free credits when you first sign in. You can purchase more via in-app purchase.

Q: What happens if I delete the app?

A: All local data will be deleted. Your account and credits are saved on our servers — sign in again to restore them.

Q: Which languages are supported for OCR?

A: Primarily Chinese (Simplified) and English. Apple Vision Framework can also auto-detect other languages.

Q: How do I get my Support Code?

A: Go to User Center in the app. Your Support Code is displayed there.

4. Troubleshooting

OCR not working: Ensure Photo Library access is granted. Try with a clearer image.

LLM features unavailable: Check your internet connection and ensure you have sufficient credits.

Search not finding results: Try different keywords. Ensure you have saved notes.
