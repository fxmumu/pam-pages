常见问题与快速解决

1) 无法加载原图 / 出现 PHPhotosErrorDomain 3303

这种情况常见于照片存储在 iCloud 而设备未下载完整原件的情形。排查步骤：

  • 确保设备网络连通并允许应用使用网络（Wi‑Fi / 蜂窝）。
  • 在系统“设置 → 照片”中查看是否启用了“优化 iPhone 存储”；如需要，临时切换为“下载并保留原件”以测试。
  • 在 App 内长按/打开该照片并在系统照片应用中查看是否可下载原始文件。
  • 开发者提示：若您是开发者，请检查照片请求 API 中是否允许网络访问（PHImageRequestOptions.isNetworkAccessAllowed）。

2) 图片缩略图 / 预览不显示或质量低

  • 这是由于请求的目标尺寸或缓存策略决定的；退出并重新打开相册有时可触发刷新。
  • 在“设置 → 隐私与安全 → 照片”中确认本应用被授予了照片访问权限。

3) 购买/订阅问题（购买失败 / 恢复失败 / 收据相关）

  • App 内购买通过 Apple App Store 完成，购买和退款请先查看 Apple 的订单与收据信息。
  • 要恢复购买：在 App 的设置页使用“恢复购买”按钮（或在 App Store 的购买记录中查找订单）。
  • 如果仍有问题，请把购买凭证截图或 App Store 订单 ID 发给我们，我们将协助核查。

4) 条款 / 隐私页面未加载

应用内条款尝试先加载内置的 `Docs/terms.html`（若已打包），若未打包则回退到在线托管页面。若您在设备上看不到条款，请确认应用是否包含该 HTML 文件（开发者需在 Xcode 的 Build Phases → Copy Bundle Resources 中添加）。

收集日志与诊断信息

在排查复杂问题时，日志非常有帮助。以下为常用方法：

  • 使用 Console.app（通过 macOS 的“控制台”应用）连接设备并重现问题，导出筛选后的日志。
  • 在 iOS 上，可按音量 + 电源触发屏幕截图并使用系统诊断，但请仅在确认没有敏感信息时再发送给我们。
  • 开发者可在 Xcode 中运行 App 并查看控制台输出： 
    xcodebuild -project PhotoAlbumManager.xcodeproj -scheme PhotoAlbumManager -configuration Debug -sdk iphonesimulator -destination 'platform=iOS Simulator,name=iPhone 16' build
